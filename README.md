# 🌐 TCP Congestion Control Analysis using NS2

## 📌 Overview

This project focuses on the simulation and analysis of TCP Congestion Control mechanisms using the Network Simulator 2 (NS2). The objective is to study how TCP manages network congestion and maintains efficient data transmission under varying network conditions.

---

## 🎯 Objectives

* Understand TCP Congestion Control algorithms.
* Analyze network performance under congestion.
* Measure throughput, packet loss, and delay.
* Evaluate TCP behavior during congestion avoidance and recovery.

---

## 🛠 Technologies Used

* NS2 (Network Simulator 2)
* TCL Scripting
* Networking Concepts
* Linux Environment

---

## 📊 Features

✔ TCP Traffic Simulation

✔ Congestion Analysis

✔ Throughput Evaluation

✔ Packet Loss Measurement

✔ Network Performance Visualization

✔ Simulation Result Analysis

## 📈 Parameters Analyzed

* Packet Loss
* End-to-End Delay
